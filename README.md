# AquaLand - Water Theme Park Website

> A static website for a fictional water theme park (team assignment)

**Tech Stack:** HTML · CSS · JavaScript
---
🌊 **[Open Website](https://dina-khan.github.io/aqualand/)**

📄 **[Report](docs/report.pdf)**

---

## Pages

- **Home** - animated news ticker, CSS image slideshow, rides preview, 
  social media section, gradient animations
- **Rides** - 2×4 CSS grid with hover rotation animation for 8 rides
- **Tickets** - ticket preview and booking section

## Features
- Animated scrolling news ticker and wave animation (CSS keyframes)
- Auto-playing CSS image slideshow (5 slides)
- Responsive footer navigation
- Custom fonts loaded via `@font-face`
- CSS grid layouts and gradient backgrounds throughout

## Run
Open `index.html` in a browser - no server needed.
